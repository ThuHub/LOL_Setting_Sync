# 英雄联盟客户端个性化设置

#### 一、项目说明

该项目为本人按照自己的游戏习惯将英雄联盟客户端键位\&选项设置调校后生成的JSON配置文件，可在任意终端下载以实现异地游戏一键覆盖设置从而保持手感的目的。

#### 二、配置说明

该配置文件由英雄联盟客户端自动生成，文件路径为：“X:\\Program Files (x86)\\英雄联盟\\Game\\Config”



#### 三、生成方法

1. 开始一场对局
2. 按ESC键对游戏项进行设置
3. 设置完成后保存
4. 退出游戏
5. 打开游戏安装目录中的Config文件夹（示例：“X:\\Program Files (x86)\\英雄联盟\\Game\\Config”）
6. PersistedSettings.json即为刚才的设置配置文件
7. 右键此文件，将只读属性打勾并保存
8. 将该文件网络备份

#### 四、使用方法

1. 在新设备上下载提前备份好的配置文件
2. 将配置文件覆盖粘贴到新设备的同名文件夹下（检查只读属性是否勾选）
3. 登录游戏，各项设置即可恢复到备份时的状态

## *无论你在哪里，设置始终如一*
